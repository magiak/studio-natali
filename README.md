# Studio Natali

Website for **Studio Natali** — kadeřnictví a nehtový salon v Říčanech u Prahy.

## About

Studio Natali nabízí profesionální péči o vlasy a nehty v přívětivém prostředí.

- **Address:** Černokostelecká 80/42, 251 01 Říčany u Prahy
- **Phone:** 774 889 606
- **Facebook:** [Studio Natali](https://www.facebook.com/profile.php?id=100054553498498)

## Tech Stack

Single-page static website built with:

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (scroll effects, mobile menu, intersection observer)
- Google Fonts (Playfair Display, Inter)

No build tools or dependencies required.

## Hosting

- Hosted on **GitHub Pages** from the `main` branch
- Repository: [magiak/studio-natali](https://github.com/magiak/studio-natali)
- Custom domain: **natali.lukaskmoch.cz** (configured via `CNAME` file)

## Local Development

Open `index.html` directly in a browser — no server or build step needed.
